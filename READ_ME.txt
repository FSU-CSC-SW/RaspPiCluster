"main.py" is the main file of the application. To run the program, just run this file

files that have "Func" in the name, are computational files and thus dont need to be modified. this includes the "AlgCycle_1.py" file as well.

"CoreRecorder.py" is the file that runs the functions, records the runtime of them, and records the data as a string

"File_Manager.py" is the file that handles info-dumping the data to a text file

"Prompts.py" is the file responsible for what is written as prints statements to the IDLE and text file

"test_data.py" is the file that contains the data that the computational functions are tested by

"READ_ME.txt" is the documentation file to help guide the user's understanding of the application.
